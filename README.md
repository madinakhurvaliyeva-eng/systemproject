# Madinaga Auth

Team-based authentication coursework project built with Express and a simple JSON data store.

## Features

- User registration
- User login with JWT
- Forgot password flow
- Password reset with token
- Demo accounts for presentation

## Tech Stack

- Node.js
- Express
- bcryptjs
- JSON file storage
- Vanilla HTML, CSS, and JavaScript

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
